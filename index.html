Keyboard Shortcuts

  • 'CTRL F' to find information on the webpage
  • 'Win(windows key)' + V [this will open the copy/paste clipboard]
  • 'CTRL' + H [this will open the browser history]
  • 'CTRL' + X [this cuts the selected text from an input field and places it in the clipboard]
  • 'CTRL' + R [this refreshes the page you're currently on]
